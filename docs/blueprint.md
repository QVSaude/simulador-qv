# **App Name**: Health Plan Quotes

## Core Features:

- Home Page with Quote Form: Landing page with a form to collect user data: Full Name, Phone (WhatsApp), Age, City/State, Plan Type (Individual, Family, Business), Number of People, Has CNPJ? (Yes/No).
- WhatsApp Integration: Upon submitting the form, the app validates required fields, generates an automatic message with the filled data, and redirects the user to WhatsApp with the pre-filled message.
- Configurable WhatsApp Number: Admin functionality to set the WhatsApp number directly in the code.
- Confirmation Message: Display a confirmation message before opening WhatsApp.
- WhatsApp Floating Icon: A persistent WhatsApp icon to allow the user to start a chat anytime.
- Basic SEO: Basic search engine optimization (SEO) implementation (title, description).

## Style Guidelines:

- Primary color: Soft blue (#A7D1FF) to inspire confidence and reliability in health-related matters.
- Background color: Light blue (#EAF4FF), a desaturated version of the primary color to provide a calm backdrop.
- Accent color: A gentle green (#B2FFB2), analogous to blue, symbolizes health and vitality. This color will provide clear contrast and make important elements like call to action buttons stand out.
- Headline font: 'Poppins' (sans-serif) for a clean and contemporary appearance.
- Body font: 'PT Sans' (sans-serif), offering readability and a touch of warmth, suitable for all text to give the site a professional look.
- Use icons related to health plans to help communicate the service.
- Clean layout with focus on the quotation form.
- Subtle animations on button hovers and form submissions to enhance user experience.
